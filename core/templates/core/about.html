Bagaimana ElectroMatch Bekerja?

Sistem kami menggunakan algoritma SAW (Simple Additive Weighting) untuk menghasilkan rekomendasi yang objektif dan akurat.

1
input

Input Preferensi

Pengguna memasukkan kategori, budget, dan kebutuhan spesifik perangkat.

2
database

Ambil Data Produk

Sistem mengambil semua data produk beserta nilai kriteria SAW dari database.

3
calculate

Hitung SAW

Algoritma SAW menghitung skor setiap produk berdasarkan bobot kriteria yang ditentukan.

4
leaderboard

Tampilkan Hasil

Produk diurutkan dari skor tertinggi sebagai rekomendasi terbaik untuk pengguna.
